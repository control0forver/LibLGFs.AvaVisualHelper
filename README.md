# LibLGFs.AvaVisualHelper

### Install
Add to your project:
```xml
<ItemGroup Condition="'$(Configuration)' == 'Release'">
  <PackageReference Include="LibLGFs.AvaVisualHelper.Release" Version="1.0.0" />
</ItemGroup>
<ItemGroup Condition="'$(Configuration)' == 'Debug'"> 
  <PackageReference Include="LibLGFs.AvaVisualHelper.Debug" Version="1.0.0" />
</ItemGroup>
```
